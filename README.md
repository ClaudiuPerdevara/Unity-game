## 📌 Overview
This is an interactive 3D demo game developed using the **Unity Engine** and **C#**. The game combines classic adventure and exploration elements with an educational twist, requiring the player to solve trivia questions to progress through the environment.

## 🗺️ Gameplay & Levels

* 🌲 **Level 1: The Forest Exploration**
  The player is dropped into a sprawling 3D forest environment. The objective is to locate 10 hidden pages scattered throughout the map. Finding a page triggers a question prompt. After successfully answering all questions, the player must locate an NPC to receive the key to the next area.

* 🗝️ **Level 2: The Dungeon**
  A darker, enclosed environment where the player must navigate through a dungeon layout. The progression relies on finding keys and chests. Similar to the first level, interacting with the chests introduces trivia challenges that must be solved to unlock the rewards.

## 🛠️ Tech Stack
* **Game Engine:** Unity 3D
* **Programming Language:** C#
* **Mechanics:** 3D collision detection, UI/Canvas scripting, NPC interaction, basic inventory/key management.
